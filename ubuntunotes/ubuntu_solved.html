疑难解决¶

删除软件（usplash）失败¶

  1. 找到dpkg数据文件

    gvim /var/lib/dpkg/status

  2. mp3转码

    sudo apt-get install convmv iconv python-mutagen
find . -iname "*.mp3" -execdir mid3iconv -e GBK {} \;

  3. 修改相关项目的 Status 字段。 [1]

    Status: purge ok not-installed

修改登录画面（lightdm）¶

  1. 查找配置文件

    /etc/lightdm/unity-greeter.conf

  2. 修改 background 项目

用户home目录下的默认目录改成英文¶

export LANG=en_US
xdg-user-dirs-gtk-update

删除多余的内核¶

  1. 使用aptitude、synaptic删除多余内核

  2. 更新GRUB，查看残留内核并移除

    sudo update-grub

  3. 再更新GRUB

  4. 更新BURG(如果安装)

    sudo update-burg

xterm解决中文问题¶

  1. 写配置文件

    ! English font
xterm*faceName: DejaVu Sans Mono:antialias=True:pixelsize=14
! Chinese font
xterm*faceNameDoublesize: WenQuanYi Micro Hei:pixelsize=14

  2. 加载配置文件

    xrdb -load .Xdefaults
