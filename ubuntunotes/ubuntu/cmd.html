1. 命令总结¶

1.1. 挂载ISO到光盘¶

  • mount 挂载命令
  • umount 卸载命令
  • -o loop 指gcc-4.6-base, libc-bin, libc6, libgcc1, multiarch-support定选项,挂载loop设备 [1] 。默认是只读
sudo mount -tiso9660 -o loop /home/matt/precise-desktop-i386.iso /cdrom
sudo mount -tiso9660 -o loop,rw /home/matt/precise-desktop-i386.iso /cdrom
sudo umount /cdrom

1.2. 解压gz、tgz、bz2¶

  • bz2比gz 更小
tar xvzf file.tar.gz
tar jvzf file.tar.bz2
tar xvzf file.tgz

1.3. 向sources.lst添加光盘源¶

sudo apt-cdrom -d /cdrom -m -o=Dir::Media::MountPath=/cdrom add

1.4. 查询软件包包含的文件¶

dpkg -L pkgname

1.5. 启动更新管理器¶

update-manager

1.6. 启动更新服务器（升级）¶

update-manger -d

1.7. 检索历史命令(显示行号)¶

  1. cat

    cat -b ~/.bash_history | grep ls
!123

  2. history

    history | grep ls

  3. alias

    alias searchistory="history | grep "

1.8. 按日期备份¶

tar cPizvf backup-`date +%Y%M%d-%S`.tar.gz /var/cache/apt/archives --exclude=/var/cache/apt/archivesapartial/* --exclude=/var/cache/apt/archives/lock
[1]The Linux loop device can be used to handle this - it makes a file look like a filesystem, and the file can be defined as a byte offset from the beginning of a primary partition
